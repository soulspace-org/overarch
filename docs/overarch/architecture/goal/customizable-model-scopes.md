
# Customizable Scopes (Goal)
## Description
The scope of the viewpoints on the model is selectable.


## Documentation
The scope defines which elements of the model are considered internal
and which are considered external in the viewpoints. Selecting the scope allows
to focus on a specific part of the model and to hide the rest in the context of
viewpoints and the generation of artifacts.
         
By supporting customizable model scopes, models can be created on an enterprise
level and still be views from the scope of e.g. a single system.
